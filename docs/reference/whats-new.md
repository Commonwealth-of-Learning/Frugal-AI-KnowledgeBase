# Knowledge base updates

This page will record what changes in the knowledge base itself: new guides, new component cards, and structural changes, so a returning reader can see what is new since their last visit.

It is under development. It is a record of changes to these pages, not a news feed: for Frugal AI programme news, strategy, and events, [COL news](https://www.col.org/news/) is the source, as [From the Commonwealth of Learning](../README.md#from-the-commonwealth-of-learning) sets out.

## Planned contents
