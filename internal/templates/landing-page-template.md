# Landing Page Template (retired as a separate scaffold)

The landing page is a single page with a single live exemplar: `docs/README.md`. Its section order is owned by `internal/editorial-guide.md` (Page Patterns — Landing Pages), including what a landing page must not contain (shell commands, custom code, long setup instructions, catalogues of pages that do not exist yet).

Edit `docs/README.md` against that pattern directly; no separate scaffold is maintained.
