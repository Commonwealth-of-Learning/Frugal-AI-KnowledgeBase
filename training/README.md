# Training materials

This section will hold the training materials for the Frugal AI programme: facilitator guides, workshop agendas, exercises, and assessment ideas for teaching institutions to build and operate the stack.

It is under development. Until it fills in, the Frugal AI knowledge base itself is the self-paced training path: the guides are tiered from Start here through Build further to Advanced, each with a level label and an expected time, and the coding agent guide doubles as the capacity-building exercise.

## Planned contents

- Facilitator guides for the tiered build path, from the first chat service to the coding agent.
- Workshop agendas and timing for half-day and full-day sessions.
- Exercises and checkpoints matched to the guides' verification steps.
- Localisation notes for running training in local languages and contexts.
